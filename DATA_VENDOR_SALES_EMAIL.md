# 数据源询价邮件模板

主题：申请投研/量化数据 API 试用与报价

您好，

我正在搭建一个面向 A股和美股的本地投研与回测系统，希望评估贵司数据服务的 API 授权、覆盖范围和报价。

当前需求如下：

1. A股历史行情：至少 5 年以上日线、复权因子、停复牌、涨跌停、成交额和换手率。
2. A股基本面：利润表、资产负债表、现金流量表、分红、公告索引和财报披露日期。
3. 研报与一致预期：盈利预测、目标价、机构评级、行业分类和预测更新时间。
4. 事件数据：业绩预告、订单公告、政策催化、并购重组、限售解禁和高管增减持。
5. 美股数据：历史行情、拆股分红、财务、新闻和 SEC 披露映射。
6. 供应链关系：客户、供应商、上下游公司、关系权重、数据来源和更新时间。
7. 使用场景：个人/内部投研、策略回测、Paper Trading 和投资辅助决策，不对外转售原始数据。

烦请提供：

- 可试用的数据范围和试用周期。
- API 文档、调用限制、历史数据覆盖年限和更新频率。
- 个人授权、团队授权或企业授权的报价。
- 是否允许用于本地回测、因子研究和 AI 分析。
- 是否覆盖北交所、港股和美股。
